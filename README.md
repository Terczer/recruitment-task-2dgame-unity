MAIN MENU

3 buttons: Play, Settings, Exit that can be clicked on and navigated between using the keyboard. 
The exit button shuts down the game. 
The settings button takes you to the Settings screen
Play button starts the game.


SETTINGS MENU

2 checkboxes: Checkbox 1 and Checkbox 2, no functionality besides checking them and unchecking. No data to be saved. 
Back button to go back to Main Menu
The escape button takes you also back to Main Menu.


GAMEPLAY

The camera background cleared to a green solid color. The camera at default (0, 0, -10) position. Player and enemies are rendered as sprites at scale (0.2, 0.2, 0.2) using Unityâ€™s default Circle sprite. 
Using the keyboard the player is able to control their character within the screen. 1000 enemy characters are running away from the player. Enemies are not able to leave the screen. When the player touches an enemy character the enemy stops moving. 
The escape button takes you back to Main Menu.


ADDITIONAL REQUIREMENTS

The player is able to navigate through the game using a mouse or keyboard alone. 
The UI is implemented using uGUI
